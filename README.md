## 📌 How to Use

Feel free to browse through the folders for notes, scripts, and exercises.  
All scripts are tested on recent Linux distributions (like Ubuntu, Fedora, etc.).

To run a script:

```bash
chmod +x scripts/example.sh
./scripts/example.sh
```
